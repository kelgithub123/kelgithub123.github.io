El usuario se compromete a utilizar el servicio de manera ética y responsable.

Está estrictamente prohibido utilizar el servicio para:

Cualquier intento de uso con fines de spam podrá resultar en el bloqueo inmediato del número.

Cuando una solicitud sea marcada como urgente, el usuario deberá abstenerse de enviar múltiples mensajes adicionales. El envío excesivo puede afectar la capacidad del sistema para atender otros casos prioritarios.

El usuario acepta cumplir con todas las leyes y regulaciones aplicables en su jurisdicción, incluyendo aquellas relacionadas con:
